#GitControl
This project created using [Angular CLI](https://github.com/angular/angular-cli)(v6.2.4) and [Nest.JS](https://nestjs.com)(v5.4.0)

#How to start project

First of all you need to clone project from GitHub to your local project
```
git clone git@github.com:NikitaGlukhi/GitControl_Project.git <project>

cd <project>
```
Start MongoDB
```
sudo mongod
```

Start server
```
npm run start:server
```

Start front-end
```
npm run start:frontend
```
After that go to `http://loalhost:4200`
